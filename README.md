# Calculadora

Site de uma calculadora totalmente funcional com lógica matemática, feedback visual imediato e com alternância de temas (modo claro e escuro).

## Link para o site

Hospedei gratuitamente o projeto usando Vercel, acesse pelo link que fica ao lado dos arquivos ou pelo link abaixo:

```bash
calculadora-ten-gules.vercel.app
```

### Tecnologias utilizadas

- HTML
- CSS
- JAVASCRIPT
