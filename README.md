## Funcionalidades
- **Design Responsivo** – Interface adaptável para Desktop, Tablets e Celulares.
- **Modo Dark/Light** – Alternância de tema suave com persistência visual.
- **Entrada via Teclado** – Filtro de caracteres para garantir que apenas números e operadores sejam inseridos.
- **UI Moderna** – Uso de variáveis CSS e unidades escaláveis (REM/EM) para acessibilidade.
